## 初回起動

### 1. ターミナルを開く

Macの「アプリケーション」→「ユーティリティ」→「ターミナル」

または、Spotlight検索（⌘ + Space）で「ターミナル」と検索

### 2. プロジェクトフォルダに移動

```bash
cd /Users/suzukishouta/Dev/utage-transcriber
```

### 3. サーバーを起動

```bash
npm run dev
```

### 4. 起動完了

数秒待つと、以下のようなメッセージが表示されます：

```
  ▲ Next.js 15.5.6
  - Local:        http://localhost:3000
  ✓ Ready in 2.3s
```

### 5. ブラウザで開く

ブラウザ（Chrome、Safari など）で以下のURLを開きます：

```
http://localhost:3000
```

## アプリの使い方

### ステップ1: 動画URLを入力

1. UTAGEのオンラインコースページを開く
2. 文字起こししたい動画のURLをコピー
3. アプリの入力欄にペースト
4. 「動画を検出」ボタンをクリック

### ステップ2: 文字起こし

1. 動画が検出されたら「文字起こし開始」ボタンをクリック
2. **約10〜20分待ちます**（60分動画の場合）
3. 完了すると文字起こし結果が表示されます

### ステップ3: テキストをコピー

- 「コピー」ボタンで文字起こしテキストをコピーできます
- NotionやGoogleドキュメントなどに貼り付けて保存できます

### ステップ4: 要約を生成（オプション）

- 「要約を生成」ボタンをクリック
- AIが自動で3〜5行の要約を作成します
- 重要なポイントが箇条書きで抽出されます

## トラブルシューティング

### Q. `http://localhost:3000` にアクセスできない

**A.** サーバーが起動しているか確認してください。

ターミナルで以下のメッセージが表示されているか確認：

```
✓ Ready in 2.3s
```

表示されていない場合は、もう一度 `npm run dev` を実行してください。

### Q. 文字起こしが途中で止まる

**A.** 以下を確認してください：

1. **インターネット接続**を確認
2. **OpenAI APIの残高**を確認（https://platform.openai.com/usage）
3. ターミナルにエラーメッセージが出ていないか確認

### Q. 「動画を検出できません」と表示される

**A.** 動画URLが正しいか確認してください。

正しいURL例：
```
https://s3.ap-northeast-1.wasabisys.com/utagesystem-video/xxxxx/xxxxx/video.m3u8
```

### Q. ポート3000が既に使われていると言われる

**A.** 別のアプリがポート3000を使っている可能性があります。

1. ターミナルで `Ctrl + C` を押して既存のサーバーを停止
2. もう一度 `npm run dev` を実行

それでもダメなら：

```bash
lsof -ti:3000 | xargs kill -9
npm run dev
```

## 注意事項

### 💰 料金について

- このアプリはOpenAI APIを使用します
- 60分の動画で約**$0.50〜$1.00**（約75〜150円）の料金がかかります
- OpenAIアカウントの残高を定期的に確認してください

### ⏱️ 処理時間について

- 60分の動画の場合、約**10〜20分**かかります
- 処理中はブラウザタブを閉じないでください
- PCをスリープモードにしないでください

### 💻 動作環境

- このMac（suzukishoutaのMac）でのみ使用可能
- サーバーを起動している間だけ使えます
- ターミナルを閉じるとサーバーも停止します

### 🔐 セキュリティ

- APIキーは `.env.local` ファイルに保存されています
- このファイルはGitHubにアップロードされません（安全です）
- 他人とAPIキーを共有しないでください

## よくある質問

### このアプリを他のPCで使いたい

1. GitHubからコードをクローン
2. `.env.local` ファイルを作成してAPIキーを設定
3. `npm install` でパッケージをインストール
4. `npm run dev` で起動

### 同時に複数の動画を処理したい

現在のバージョンでは1つずつ処理してください。同時処理には対応していません。

### 文字起こし結果を保存したい

「コピー」ボタンでコピーして、Notion、Googleドキュメント、メモアプリなどに貼り付けて保存してください。
